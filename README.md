**Real-time American Sign Language (ASL) alphabet recognition** using **MobileNetV2** transfer learning and deep learning. Built for the **Samsung SFT Innovation Competition**.

---

## 📊 Overview

This project enables real-time recognition of 26 ASL alphabet signs (A-Z) from webcam input. Using a MobileNetV2-based architecture with transfer learning, the model achieves **95%+ accuracy** on standard test sets and runs efficiently on laptop CPUs.

### 🎯 Key Features

- ✅ **Real-time detection** via webcam (10-15 FPS on CPU)
- ✅ **MobileNetV2** architecture for fast, lightweight inference
- ✅ **Data augmentation** for robust real-world performance
- ✅ **Interactive learning** mode to collect feedback and improve
- ✅ **Top-3 predictions** with confidence scores
- ✅ **Live accuracy testing** with keyboard feedback
- ✅ **Fine-tuning support** for custom data
